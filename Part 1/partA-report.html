Formula used for updating belief states:

  • s = old state
  • s' = new state
  • b = old belief state, and b(s) probability of s given belief state b
  • a = action
  • b’ = new belief state
  • b’(s’) = probability of s’ given b’
  • o = observation
  • P(o | s', a) = Probability of observing o given we reach s' by action a
  • T(s, a, s') = Transition probability for reaching s' from s by action a
  • P(o | a, b) = Probability of observing o given action a and old belief b
